# 🛫 MF-UAV

> 一个基于cesium框架的无人机控制模拟案例, 服务地址: https://fengtianxi001.github.io/MF-UAV/

## 项目简介

项目主要做了如下内容:

- [x] 通过按键调整飞行器的飞行姿态
  
- [x] 实现全键无冲
  
- [x] 实时获取飞行器的飞行参数并在面板展示
  
- [x] 通过飞行器的经纬度坐标完成leaflet的轨迹定位
  
- [x] 在`Vue3`框架下对`cesium`封装的探索
